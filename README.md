# Task Manager API (CRUD)

This is a simple backend project built using FastAPI.

## Features

* Create Task
* View Tasks
* Update Task
* Delete Task

## Tech Stack

* Python
* FastAPI
* Uvicorn

## How to Run

```bash
pip install fastapi uvicorn
python -m uvicorn main:app --reload
```
